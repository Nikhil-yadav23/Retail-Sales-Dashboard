# Retail-Sales-Performance-Dashboard
## Overview
This project is an interactive dashboard built using SQL and Power BI to analyze retail sales performance.

## Tools & Technologies
- SQL
- Power BI
- Excel

## Techniques Used
- Data Cleaning and Preprocessing
- KPI Analysis (Revenue, Profit, Orders, Profit Margin)
- Time Series Analysis
- Regional Performance Analysis
- Data Visualization

## Key Insights
- West region generated highest sales
- Sales increased during year-end period

- Dashboard (.pbix)
